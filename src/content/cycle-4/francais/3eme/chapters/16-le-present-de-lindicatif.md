# Le Présent de l'Indicatif

<tense-values-svg></tense-values-svg>

<callout type="info" title="Introduction">
Le présent de l'indicatif est le temps le plus utilisé en français. Il sert à exprimer une action qui se déroule au moment où l'on parle, mais il a aussi d'autres valeurs fondamentales pour le Brevet. Maîtriser sa conjugaison pour tous les groupes est indispensable.
</callout>

### 1. Conjugaison : Les terminaisons ✍️

Le français divise les verbes en 3 groupes. Chaque groupe a ses propres terminaisons au présent.

*   **1er groupe (-er)** : -e, -es, -e, -ons, -ez, -ent.
    *   *Exemple (Chanter)* : Je chant**e**, tu chant**es**, il chant**e**, nous chant**ons**, vous chant**ez**, ils chant**ent**.
*   **2ème groupe (-ir / -issant)** : -is, -is, -it, -issons, -issez, -issent.
    *   *Exemple (Finir)* : Je fin**is**, tu fin**is**, il fin**it**, nous fin**issons**, vous fin**issez**, ils fin**issent**.
*   **3ème groupe (Tous les autres)** : Souvent -s, -s, -t, -ons, -ez, -ent.
    *   *Verbes en -dre* (sauf -indre) : -ds, -ds, -d, -ons, -ez, -ent.
    *   *Pouvoir / Vouloir* : -x, -x, -t, -ons, -ez, -ent.

### 2. Les auxiliaires Être et Avoir 👑

Ils sont irréguliers et indispensables pour former les temps composés.

| Personne | Être | Avoir |
| :--- | :--- | :--- |
| Je / J' | suis | ai |
| Tu | es | as |
| Il / Elle | est | a |
| Nous | sommes | avons |
| Vous | êtes | avez |
| Ils / Elles | sont | ont |
### 3. Les valeurs du Présent 💡

Au Brevet, on te demandera souvent "Quelle est la valeur du présent dans cette phrase ?". Voici les 4 valeurs à connaître :

1.  **Présent d'énonciation** : L'action se passe maintenant. (*Je lis cette fiche.*)
2.  **Présent de vérité générale** : C'est toujours vrai. (*La Terre est ronde.*)
3.  **Présent d'habitude** : Une action qui se répète. (*Je mange à midi tous les jours.*)
4.  **Présent de narration** : On utilise le présent dans un récit au passé pour rendre l'action plus vive. (*Il marchait, soudain il tombe.*)
